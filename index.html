Single Points of Failure Identified

dimension_salesforce_users — 30+ downstream DAGs depend on this single dimension. A delay in the Salesforce Fivetran sync at 07:15 UTC cascades across nearly every support, fraud, and QA pipeline. This is the primary cause of the dimension delivery delays.


fact_salesforce_amazon_connect_contact_trace (Fivetran) — The sole upstream source for Amazon Connect CTR data feeding routing profiles, surveys, and SLA metrics. Any Fivetran sync delay or schema drift causes missing data in Amazon Connect.


Sprout Social / Persona API calls — Direct REST API calls via PythonOperator. No retry circuit breaker, no fallback cache. API rate limits, token expiry, or 3rd party outage = full DAG failure with no partial recovery.
